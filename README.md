# live-link
A video conferencing platform using GraphQL, Redis, Express, React, &amp; Material-UI.

## How to Run
1. Create a .env file in root directory
```
JWT_SECRET=
```

2. Pull node 18 docker image
```
docker pull node:18
```

3. Build
```
docker-compose build
```

4. Run
```
docker-compose up
```
